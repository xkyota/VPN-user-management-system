# Requirements Review

## Functional requirements

- User registration or user creation by administrator.
- Login system.
- User list page.
- User details page.
- Edit user data.
- Delete or deactivate user.
- VPN access status: active, blocked, expired.
- Expiry date for VPN access.
- VPN configuration file field.
- Activity history / logs.
- Admin dashboard.

## Non-functional requirements

- The app must be easy to run.
- The app must use Docker.
- Database: PostgreSQL.
- Source code must be stored in GitHub.
- Basic validation must be included.
- Errors must be shown clearly.
