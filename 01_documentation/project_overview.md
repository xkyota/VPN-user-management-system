# VPN User Management System

## Project Description

The VPN User Management System is a web application designed to help administrators manage VPN users in one place. The system will make it easier to create users, control their access, store account expiry dates, manage VPN configuration files, and review system activity logs.

The project is developed as part of a 160-hour practice assignment. The work includes planning, implementation, testing, documentation, and final preparation for review.

## Project Goals

- Create a simple and clear web interface for VPN user administration.
- Store VPN user information in a structured database.
- Allow administrators to activate, deactivate, update, and delete users.
- Track user expiry dates and access status.
- Store or link VPN configuration files for each user.
- Keep logs of important actions for basic auditing.
- Run the application in a Docker-based environment.

User roles:

- Administrator: manages users, access, configuration files, and logs.
- Standard user: has limited access to personal VPN-related information if this feature is implemented.

## Main Features

- User list with basic search or filtering.
- Add, edit, and delete VPN users.
- Access status management.
- Expiry date management.
- VPN configuration file management.
- Action log overview.
- Docker Compose setup for local development.

## Expected Result

The final result should be a working web application with source code, database structure, Docker configuration, testing evidence, screenshots, weekly reports, and final documentation. The project should be clear enough for another developer or supervisor to understand how it works and how to run it.
